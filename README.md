** Skill Gap Analysis & Prediction using Streamlit **

A machine learning-based web application to identify and analyze skill gaps among students or employees.
Built with Python (Scikit-learn, Pandas) and deployed using Streamlit Cloud for an interactive experience.

Features

  Upload Excel dataset (.xlsx) from sidebar
  
  Automatic preprocessing and encoding of data
  
  Real-time model training (Random Forest Classifier)
  
  Shows validation accuracy and classification report
  
  Quick demo prediction interface
  
  Simple, responsive, and easy-to-use UI
